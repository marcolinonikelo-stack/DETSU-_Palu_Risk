# DETSU Palu Risk v1.1 🇹🇬

**Objectif** : Prédire le risque de paludisme à Lomé selon pluie et température. 
**Cible** : Projet IA Santé pour KAIST 2029

### Comment utiliser
1. Lancer `detsu_v1_1.py` 
2. Entrer : Quartier, Pluie en mm, Température en °C
3. Lire le niveau de risque + conseil santé

### Version
- **v1.0** : Risque selon pluie seule
- **v1.1** : Ajout température + condition double `and`
