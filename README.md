# DETSU Palu Risk v1.2 🇹🇬 `BAC 1 ADMIS ✅ | KAIST 2029 : 1181 jours`

**Objectif** : Prédire le risque de paludisme à Lomé selon pluie et température.
**Cible** : Projet IA Santé pour Bourse GKS KAIST 2029

### Comment utiliser
1. Lancer `detsu_v1_2.py`
2. Entrer : Quartier, Pluie en mm, Température en °C
3. Lire le niveau de risque + conseil santé
4. **NOUVEAU** : Voir le compteur "Jours avant KAIST"
